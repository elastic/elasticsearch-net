UnionTag

Remarks

This tag should be used as the discriminator in switch expressions before accessing the actual encapsulated value in either Value1 or Value2.

As an alternative, the Match(Action<T1?>?, Action<T2?>?) or Match<T>(Func<T1?, T>, Func<T2?, T>) method might be used to access the encapsulated value in a safe way.
