Class GeoLocation

Namespace
Elastic.Clients.Elasticsearch

A latitude/longitude as a 2 dimensional point. It can be represented in various ways:

  • as a {lat, long} object

  • as a geo hash value

  • as a [lon, lat] array

  • as a string in "<lat>, <lon>" or WKT point formats
