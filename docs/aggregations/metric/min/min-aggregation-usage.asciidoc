== Min Aggregation Usage

=== Fluent DSL Example

[source,csharp]
----
s => s
.Aggregations(a => a
    .Min("min_commits", m => m
        .Field(p => p.NumberOfCommits)
    )
)
----

=== Object Initializer Syntax Example

[source,csharp]
----
new SearchRequest<Project>
{
    Aggregations = new MinAggregation("min_commits", Field<Project>(p => p.NumberOfCommits))
}
----
