[[fields-usage]]
== Fields Usage

=== Fluent DSL Example

[source,csharp]
----
s => s
.Fields(fs => fs
    .Field(p => p.Name)
    .Field(p => p.StartedOn)
)
----

=== Object Initializer Syntax Example

[source,csharp]
----
new SearchRequest<Project>
{
    Fields = Fields<Project>(p => p.Name, p => p.StartedOn)
}
----

[source,javascript]
.Example json output
----
{
  "fields": [
    "name",
    "startedOn"
  ]
}
----
