== Post Filter Usage

=== Object Initializer Syntax Example

[source,csharp]
----
new SearchRequest<Project>()
{
    PostFilter = new QueryContainer(new MatchAllQuery())
}
----

[source,javascript]
.Example json output
----
{
  "post_filter": {
    "match_all": {}
  }
}
----

=== Fluent DSL Example

[source,csharp]
----
s => s
.PostFilter(f => f.MatchAll())
----
