== Geo Envelope Usage

=== Fluent DSL Example

[source,csharp]
----
q
.GeoShapeEnvelope(c => c
    .Name("named_query")
    .Boost(1.1)
    .Field(p=>p.Location)
    .Coordinates(this._coordinates)
)
----

=== Object Initializer Syntax Example

[source,csharp]
----
new GeoShapeEnvelopeQuery
{
    Name = "named_query",
    Boost = 1.1,
    Field = Field<Project>(p=>p.Location),
    Shape = new EnvelopeGeoShape(this._coordinates)
}
----
