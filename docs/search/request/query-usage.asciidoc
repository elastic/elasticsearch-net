[[query-usage]]
== Query Usage

The query element within the search request body allows to define a query using the <<query-dsl,Query DSL>>.

[float]
=== Fluent DSL example

[source,csharp]
----
s => s
.Query(q => q
    .Term(p => p.Name, "elasticsearch")
)
----

[float]
=== Object Initializer syntax example

[source,csharp]
----
new SearchRequest<Project>
{
    Query = new TermQuery
    {
        Field = "name",
        Value = "elasticsearch"
    }
}
----

[source,javascript]
.Example json output
----
{
  "query": {
    "term": {
      "name": {
        "value": "elasticsearch"
      }
    }
  }
}
----
