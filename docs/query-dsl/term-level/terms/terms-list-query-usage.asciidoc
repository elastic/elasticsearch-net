== Terms List Query Usage

=== Fluent DSL Example

[source,csharp]
----
q
.Terms(c => c
    .Name("named_query")
    .Boost(1.1)
    .Field(p => p.Description)
    .DisableCoord()
    .MinimumShouldMatch(MinimumShouldMatch.Fixed(2))
    .Terms(new List<string> { "term1", "term2" })
)
----

=== Object Initializer Syntax Example

[source,csharp]
----

    Boost = 1.1,
    Field = "description",
    Terms = new List<string> { "term1", "term2" },
    DisableCoord = true,
    MinimumShouldMatch = 2
}
----
