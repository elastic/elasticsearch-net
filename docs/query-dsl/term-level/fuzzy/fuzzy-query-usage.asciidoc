== Fuzzy Query Usage

=== Fluent DSL Example

[source,csharp]
----
q
.Fuzzy(c => c
    .Name("named_query")
    .Boost(1.1)
    .Field(p => p.Description)
    .Fuzziness(Fuzziness.Auto)
    .Value("ki")
    .MaxExpansions(100)
    .PrefixLength(3)
    .Rewrite(RewriteMultiTerm.ConstantScore)
    .Transpositions()
)
----

=== Object Initializer Syntax Example

[source,csharp]
----
new FuzzyQuery
{
    Name = "named_query",
    Boost = 1.1,
    Field = "description",
    Fuzziness = Fuzziness.Auto,
    Value = "ki",
    MaxExpansions = 100,
    PrefixLength = 3,
    Rewrite = RewriteMultiTerm.ConstantScore,
    Transpositions = true
}
----
