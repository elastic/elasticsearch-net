[[fuzzy-query-usage]]
=== Fuzzy Query Usage

==== Fluent DSL example

[source,csharp]
----
q
.Fuzzy(c => c
    .Name("named_query")
    .Boost(1.1)
    .Field(p => p.Description)
    .Fuzziness(Fuzziness.Auto)
    .Value("ki")
    .MaxExpansions(100)
    .PrefixLength(3)
    .Rewrite(MultiTermQueryRewrite.ConstantScore)
    .Transpositions()
)
----

==== Object Initializer syntax example

[source,csharp]
----
new FuzzyQuery
{
    Name = "named_query",
    Boost = 1.1,
    Field = "description",
    Fuzziness = Fuzziness.Auto,
    Value = "ki",
    MaxExpansions = 100,
    PrefixLength = 3,
    Rewrite = MultiTermQueryRewrite.ConstantScore,
    Transpositions = true
}
----
