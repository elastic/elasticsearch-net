=== Ids Query Usage

==== Fluent DSL example

[source,csharp]
----
q
.Ids(c => c
    .Name("named_query")
    .Boost(1.1)
    .Values(1, 2, 3, 4)
)
----

[source,csharp]
----

    Boost = 1.1,

}
