=== Term Range Query Usage

==== Fluent DSL example

[source,csharp]
----
q
.TermRange(c => c
    .Name("named_query")
    .Boost(1.1)
    .Field(p => p.Description)
    .GreaterThan("foo")
    .GreaterThanOrEquals("foof")
    .LessThan("bar")
    .LessThanOrEquals("barb")
)
----

==== Object Initializer syntax example

[source,csharp]
----
new TermRangeQuery
{
    Name = "named_query",
    Boost = 1.1,
    Field = "description",
    GreaterThan = "foo",
    GreaterThanOrEqualTo = "foof",
    LessThan = "bar",
    LessThanOrEqualTo = "barb"
}
----
