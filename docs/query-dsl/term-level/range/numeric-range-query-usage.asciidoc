== Numeric Range Query Usage

=== Fluent DSL Example

[source,csharp]
----
q
.Range(c => c
    .Name("named_query")
    .Boost(1.1)
    .Field(p => p.Description)
    .GreaterThan(1.0)
    .GreaterThanOrEquals(1.1)
    .LessThan(2.1)
    .LessThanOrEquals(2.0)
)
----
