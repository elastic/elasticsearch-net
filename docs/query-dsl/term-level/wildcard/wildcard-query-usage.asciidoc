=== Wildcard Query Usage

==== Fluent DSL example

[source,csharp]
----
q
.Wildcard(c => c
    .Name("named_query")
    .Boost(1.1)
    .Field(p => p.Description)
    .Value("p*oj")
    .Rewrite(MultiTermQueryRewrite.TopTermsBoost(10))
)
----

==== Object Initializer syntax example

[source,csharp]
----
new WildcardQuery
{
    Name = "named_query",
    Boost = 1.1,
    Field = "description",
    Value = "p*oj",
    Rewrite = MultiTermQueryRewrite.TopTermsBoost(10)
}
----
