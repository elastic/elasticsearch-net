[[verbatim-and-strict-query-usage]]
== Verbatim And Strict Query Usage

`IsVerbatim` should be set on individual queries to take effect 

=== Fluent DSL Example

[source,csharp]
----
q
.Bool(b => b
    .Must(qt => qt
        .Term(t => t
            .Verbatim()
            .Field(p => p.Description)
            .Value("")
        ), qt => qt
        .Term(t => t
            .Field(p => p.Name)
            .Value("foo")
        )
    )
)
----

=== Object Initializer Syntax Example

[source,csharp]
