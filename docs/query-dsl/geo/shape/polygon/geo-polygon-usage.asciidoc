== Geo Polygon Usage

=== Fluent DSL Example

[source,csharp]
----
q
.GeoShapePolygon(c => c
    .Name("named_query")
    .Boost(1.1)
    .Field(p => p.Location)
    .Coordinates(this._coordinates)
    .Relation(GeoShapeRelation.Intersects)
)
----
