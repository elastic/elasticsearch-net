=== Match Bool Prefix Usage

==== Fluent DSL example

[source,csharp]
----
q
.MatchBoolPrefix(c => c
    .Field(p => p.Description)
    .Analyzer("standard")
    .Boost(1.1)
    .Query("lorem ips")
    .Fuzziness(Fuzziness.AutoLength(3, 6))
    .FuzzyTranspositions()
    .FuzzyRewrite(MultiTermQueryRewrite.TopTermsBlendedFreqs(10))
    .Name("named_query")
)
----
