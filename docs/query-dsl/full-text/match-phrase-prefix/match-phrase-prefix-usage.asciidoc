=== Match Phrase Prefix Usage

==== Fluent DSL example

[source,csharp]
----
q
.MatchPhrasePrefix(c => c
    .Field(p => p.Description)
    .Analyzer("standard")
    .Boost(1.1)
    .Query("hello worl")
    .MaxExpansions(2)
    .Slop(2)
    .Name("named_query")
)
----
