[[match-usage]]
=== Match Usage

==== Fluent DSL example

[source,csharp]
----
q
.Match(c => c
    .Field(p => p.Description)
    .Analyzer("standard")
    .Boost(1.1)
    .Query("hello world")
    .Fuzziness(Fuzziness.AutoLength(3, 6))
    .Lenient()
    .FuzzyTranspositions()
    .MinimumShouldMatch(2)
    .Operator(Operator.Or)
    .FuzzyRewrite(MultiTermQueryRewrite.TopTermsBlendedFreqs(10))
    .Name("named_query")
    .AutoGenerateSynonymsPhraseQuery(false)
)
----
