=== Common Terms Usage

==== Fluent DSL example

[source,csharp]
----
q
.CommonTerms(c => c
    .Field(p => p.Description)
    .Analyzer("standard")
    .Boost(1.1)
    .CutoffFrequency(0.001)
    .HighFrequencyOperator(Operator.And)
    .LowFrequencyOperator(Operator.Or)
    .MinimumShouldMatch(1)
    .Name("named_query")
    .Query("nelly the elephant not as a")
)
----
